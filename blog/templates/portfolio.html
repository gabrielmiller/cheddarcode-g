Rotation Encryption

This script was a learning experience on data types and object-oriented programming in Python. I made a first version and then revised it about a month later, making sure to include my early critiques. Text is entered, broken into 8 character chunks, and transformed to a multidimensional list of the ASCII binary values of each character. The multidimensional list is viewed as a square; the output is an interpretation of that same square rotated by 90 degrees. The lists are then concatenated, regrouped, and packaged back into 8 character chunks, giving an encrypted version of the original text.

Shoutwall

This site is a basic shoutwall where users enter text, with a limited selection of available HTML formatting, and it is posted to a public shoutwall. Posts are authored without a login system, based upon any name of your choosing or the user may post anonymously. This site is written in PHP and uses MySQL. MySQLi was used for the database queries.


See a live instance of it here.

Imageboard

This site is a similar site to the shoutwall but, instead of text, users upload and share images with one another. This site was built with Python and Google App Engine. The data is stored using Google Storage Engine, a NoSQL database. I used Jinja2 for the templating on this site(which I found to be nicer than django).


See a live instance of it here.

Writing critiques

I wanted another, more comprehensive project to demonstrate using MySQL and PHP together, so I put together a more complex website with a login system, user driven content, and administration. Users can sign up and post pieces of writing to be critiqued. The administrator can log in and respond to the writings and decide whether or not to publish the piece and critique. I built this site with PHP and MySQL. Again, MySQLi was used for the database queries.


See a live instance of it here.

My blog

This very website runs on Django, and was one of my earliest adventures into Python's web development ecosystem. Django provides a very nice administration system and helped to quickly prototype my site and try different models and layouts. My blog showcases dynamic blog posts that I write along with some static pages and a form that emails me. I use a third party application to handle metatags called Taggit. The database behind my blog is MySQL. The collapsing and expanding list that I used for archives on my blog was written by Kent Safranski and the AJAX requests use jQuery.

Hi there!

My name's Gabe. I'm interested in computer programming, particularly related to web development, databases, and statistics. On this site you'll be able to read about my computer interests along with any questions and answers I have.

I used to be a Geographic Information Systems(GIS) Technician. I worked with maps, spatial data, databases, and a lot of excel spreadsheets. After several months I realized that I enjoyed programming and most things website-related so I pursued software development as a career path.

I'm currently employed by Hireku, the company that wrote and supports The Resumator. You should check us out, especially if you're a business and the hiring process bothers you! We're a rapidly growing startup with an awesome company culture.

I've always considered myself a computer nerd. Initially when I began using computers I was interested in video games. Arguably it started with The Last Ninja and Zany Golf on an Apple IIGS. Later, the game Everquest consumed many days of my teen years. After far too many late nights of computer games I graduated from that phase of my life. But I never graduated from my interest in computers and the internet.

I dabbled with programming Python and PHP before college with a high school website and my curiosity, but never delved too deeply into coding. I began learning Python in earnest in college when I was learning to use ESRI's GIS software. I realized that learning Python had more potential than just expanding my GIS skills; I saw it as a personal interest as well as a career path. Open source languages and software had always appealed to me because of their widespread adoption, especially on web platforms, but I had never made a point to scratch much deeper than the surface.

I've been learning open source technologies which I've come to find extremely valuable. I consider myself an up and coming Linux user and my skill-set is pointed toward web technologies; I've worked with Python, PHP, JavaScript, HTML, CSS, MySQL, Postgres, node.js, and mongoDB. Further, at work I regularly write and debug PHP code and MySQL queries and work with HTML, Javascript, and CSS. On my own time I've taken to learning about more cutting edge projects that use node.js and Python. In the future I'd like to learn more Math and build fun and interesting applications.

On my blog you'll find more recent activity on my adventures. You can also follow my projects at github.com/gmiller2007. Thanks for reading!

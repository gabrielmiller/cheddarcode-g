Contact Tracker

This site is a tool I wrote for my company to track landowners that we contact regarding property permission for field work. We had been using pen and paper however the system was awkward and not easily shared between offices. I made a website where we can add profiles and landowners' information in order to solve our problems. This site uses PHP and MySQL, as well as Twitter bootstrap for the css and widgets, as well as a smidgen of jQuery and jQueryUI.

Stream Mapping

This site is a data visualization for information my work collects. It is an automation of paper layouts that we already are creating in an effort to save time. The site is written in PHP and uses the html5 canvas and a JavaScript mapping library called Leaflet. I used Twitter's bootstrap for css and javascript widgets. The data was stored in a postgres database, but I changed it to MySQL in order to host it on my webhost.

Rotation Encryption

This script was a learning experience on data types and object-oriented programming in Python. I made a first version and then revised it about a month later, making sure to include my early critiques. Text is entered, broken into 8 character chunks, and transformed to a multidimensional list of the ASCII binary values of each character. The multidimensional list is viewed as a square; the output is an interpretation of that same square rotated by 90 degrees. The lists are then concatenated, regrouped, and packaged back into 8 character chunks, giving an encrypted version of the original text.
